# CopyLists

> **macOS 13+** · 原生 Swift · 数据纯本地 · [许可协议](LICENSE)（非商业免费；商业使用请先邮件联系）  
> **English:** [README_EN.md](README_EN.md)

macOS 剪贴板历史工具：快捷键呼出、回车粘贴；支持图片缩略图、**本地 OCR 搜索**、**收藏长期保留**、**⌘P 预览**（改字 / 图片标注后写回剪贴板）。开源可审，无账号、无广告；默认不做联网上传（详见「隐私与安全」）。

**仓库：** [github.com/evenjxr/copyLists](https://github.com/evenjxr/copyLists) · **下载：** [Releases 最新版](https://github.com/evenjxr/copyLists/releases/latest)

## 快速开始

### 下载安装（推荐）

1. 打开 **[Releases → Latest](https://github.com/evenjxr/copyLists/releases/latest)**，下载 **`CopyLists.dmg`**。  
2. 将 `CopyLists.app` 拖入 **应用程序**。  
3. **首次打开**：Finder 中 **右键 → 打开**（未做公证时系统可能拦截，见 [常见问题](#打开提示无法验证开发者)）。  
4. 菜单栏出现图标后即可使用；**粘贴与搜索框**需开启 **辅助功能**（见 [常见问题](#粘贴无效或搜索框无法输入)）。

### 系统要求

- macOS **13** 及以上  
- Apple Silicon 与 Intel：`build_app.sh` 默认产出 **Universal** 二进制；若 Release 里只提供 arm64 包，请以该版本说明为准。

> **国内访问 GitHub 慢**：可自行镜像到 Gitee 并在 Release 提供同版本 DMG，建议附带 SHA256 或与官方 Tag 对应说明。

## 功能一览

| 能力 | 说明 |
|------|------|
| **呼出面板** | 任意界面 `⌘⇧V` 打开历史，选中后 `↵` 粘贴并回到原应用。 |
| **类型与去重** | 自动识别 URL / 邮箱 / 路径 / 代码 / 图片 / 纯文本；重复复制合并为一条并累计次数。 |
| **键盘操作** | `↑↓` 选择 · `←→` 切换标签 · `⇧↵` 纯文本粘贴 · `⌘S` 收藏 · `⌘P` 预览 · `⌘⌫` 删除 · `⎋` 关闭。 |
| **收藏** | 收藏项**不随 LRU 清空**；清空历史时仍保留；顶部可筛选「收藏」。 |
| **预览 ⌘P** | 文字可编辑，`⌘↩` 写回剪贴板；图片支持画笔 / 箭头 / 矩形，`⌘Z` / `⌘⇧Z` 撤销重做，`⌘↩` 导出标注图。图钉可让窗口置顶；多窗口右上角纵向排列，位置与尺寸会记忆。 |
| **OCR** | 系统 **Vision** 本机识别图片内文字（简繁 + 英文），结果参与搜索。 |
| **隐私** | 菜单栏 **暂停记录**；按 Bundle ID **排除应用**（常见密码管理器默认排除）；详见下表。 |
| **设置 `⌘,`** | 历史上限（20～500）、开机启动、暂停、排除列表。 |

数据目录：`~/Library/Application Support/CopyLists/`（卸载应用后可手动删除）。

## 隐私与安全

| 项目 | 说明 |
|------|------|
| **剪贴板** | 仅用于写入本地历史；**暂停**期间不记录。 |
| **网络** | 正式发行版**不含**统计、云同步或剪贴板上传；若自行改代码联网，以你构建的版本为准。 |
| **敏感 App** | 支持排除 **Bundle ID**；1Password、Bitwarden、Dashlane、LastPass 等默认在列（前台时跳过）。 |
| **OCR** | 本机识别，结果仅存本地供搜索。 |
| **辅助功能** | 用于模拟 **`⌘V` 粘贴** 与搜索框焦点；系统设置中可随时关闭。 |
| **登录项** | 仅用于启动 CopyLists 自身。 |

## 快捷键

**主面板**

| 按键 | 动作 |
|:---:|:---|
| `⌘⇧V` | 唤起 / 隐藏 |
| `↑` / `↓` | 选择条目 |
| `←` / `→` | 切换类型标签 |
| `↵` | 粘贴 |
| `⇧↵` | 纯文本粘贴 |
| `⌘S` | 收藏 / 取消收藏 |
| `⌘P` | 预览浮窗 |
| `⌘1`～`⌘9` | 快速粘贴第 1～9 条 |
| `⌘⌫` | 删除选中项 |
| `⎋` | 关闭面板 |

**预览浮窗内**

| 按键 | 动作 |
|:---:|:---|
| `⌘↩` | 保存并关闭（文字或带标注图片写回剪贴板） |
| `⌘Z` / `⌘⇧Z` | 撤销 / 重做（图片标注） |
| `⎋` | 关闭浮窗 |

## 从源码构建

```bash
git clone https://github.com/evenjxr/copyLists.git && cd copyLists
bash build_app.sh          # 生成 CopyLists.app（项目根目录）
```

## 常见问题

### 打开提示「无法打开，因为无法验证开发者」

Gatekeeper 会拦截未在 App Store 分发、且未 **公证** 的应用。

1. **推荐**：Finder 中对 `CopyLists` **Control + 点按** → **打开** → 再点 **打开**。通常只需一次。  
2. 若无「打开」：**系统设置 → 隐私与安全性** → 找到被拦截提示 → **仍要打开**。  
3. **长期**：使用 Apple 开发者账号做 **Developer ID 签名 + 公证**，可减少新用户门槛。

### 粘贴无效或搜索框无法输入

需 **辅助功能** 权限：**系统设置 → 隐私与安全性 → 辅助功能** → 解锁 → 打开 **CopyLists**。若列表中没有，先启动应用并试一次粘贴，按系统提示授权。

### 开机启动未生效

**系统设置 → 通用 → 登录项与扩展** → 确认 **CopyLists** 已开启；若无，在应用内设置里重新开关一次「开机自动启动」。

### 历史记录在重启后丢失

数据在 `~/Library/Application Support/CopyLists/`。若异常，可尝试使用 **DMG 安装版** 而非仅 `swift run` 的调试方式。
